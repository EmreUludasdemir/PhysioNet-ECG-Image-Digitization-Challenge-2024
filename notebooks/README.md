# Jupyter Notebooks

This directory contains Jupyter notebooks for exploratory data analysis, experimentation, and visualization.

## Notebooks

### 01_eda.ipynb
Exploratory Data Analysis notebook covering:
- Data loading and inspection
- ECG image visualization
- Signal analysis
- Data quality assessment

### 02_preprocessing.ipynb
Preprocessing experiments including:
- Image preprocessing techniques
- Rotation correction evaluation
- Contrast enhancement comparison
- Noise reduction methods

### 03_model_training.ipynb
Model training experiments:
- Model architecture comparisons
- Hyperparameter tuning
- Training visualization
- Loss curve analysis

### 04_evaluation.ipynb
Model evaluation and analysis:
- SNR calculation
- Per-lead performance analysis
- Error analysis
- Visualization of predictions vs ground truth

## Usage

1. Install Jupyter:
```bash
pip install jupyter
```

2. Start Jupyter server:
```bash
jupyter notebook
```

3. Open any notebook and run cells sequentially.

## Requirements

All notebooks require the dependencies listed in `requirements.txt` to be installed.
